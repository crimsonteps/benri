# Benri 1.0.0

Benri's first public release turns frequently reused text into a keyboard-first macOS workflow. Open the panel from anywhere, find a record, and paste it back into the app you were using.

## Highlights

- Configurable global launcher and menu bar controls
- Categories, title search, inline editing, and automatic saving
- Direct copy-and-paste workflow with a safe clipboard-only fallback
- Local AES-256-GCM encrypted storage with no network requests or telemetry
- Light, dark, reduced-transparency, and Liquid Glass-aware appearance
- Universal 2 build for Apple Silicon and Intel Macs running macOS 13 or later

## Install

Download `Benri-v1.0.0-macOS-universal.zip`, move `Benri.app` to `/Applications`, and open it. Automatic paste requires Accessibility permission; copying works without it.

This community build is ad-hoc signed and is not notarized. macOS may require you to Control-click Benri, choose **Open**, and confirm once from **System Settings → Privacy & Security**.

The accompanying `.sha256` file can be verified with:

```bash
shasum -a 256 -c Benri-v1.0.0-macOS-universal.zip.sha256
```

See the [README](https://github.com/crimsonteps/quick-vault#readme) for usage, privacy details, and source build instructions.
